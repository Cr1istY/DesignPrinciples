#### 设计模式前言

作为一个有追求的程序员和完美主义者，我们理应将我们的代码达到我们所能达到的最佳水平。

同时，对于大部分程序员来说，我们可能常常受困于视野，往往忽视了对于整个软件系统的架构设计。

因此，学习设计模式，是有必要的，并且，是成长的必要之路。
